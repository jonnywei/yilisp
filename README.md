# yilisp
a small lisp language
## DataType
1. Number

   eg. 1,400
   
2. Boolean 
    
    eg. #t,#f
    
3. Symbol

    eg. var, some-var
    
4. Function 

    eg. (lambda (x) (+ x 1))
   
5. List

    eg. (1,2,3) ( a list of symbol)
    
## Syntax

## Special Forms

1. define
2. lambda
3. if
4. quote

## Builtin Functions

1. Arithmetic functions (+, -, *, /, %, =, >, <)
2. List functions (list, cons, car, cdr)
3. IO functions (println, now)
